# AIEvent_0911
0911開催のイベントのための教材置き場です

## Cursorの起動

### 1. Cursorをインストール

- https://cursor.com/ja

### 2. 教材のファイルをGithubからダウンロード

- github

### 3. 教材をCursorで開く

- 教材のzipファイルを解凍
- Cursorを開く
- ファイル＞フォルダーを開く…から、「site」フォルダを開く

### 4. 教材をプレビュー

- siteフォルダを開く
- 「index.html」を開く
- 顧客管理画面が開けばOK

### 5. Cursorチャットの使い方

- 右上のアイコンを押します
- チャットが出てきます
